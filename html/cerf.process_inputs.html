class cerf.process_inputs.process_expansion.ProcessExpansion(expansion_plan_file, scenario, target_yr, target_techs, tech_dict, primary_zone_dict, expansion_plan_xml_dir, output_csv=False)[source]¶

Bases: object

Convert expected electricity capacity plan to CERF XML format.

Parameters

  • expansion_plan_file – Full path with file name and extension of input expansion plan file.

  • scenario – Name of the target scenario.

  • target_yr – Four digit year to extract.

  • target_techs – List of technologies to include in the expansion plan.

  • tech_dict – Dictionary of {cerf_technology_name: cerf_technology_id}

  • primary_zone_dict – Dictionary of {cerf_state_abbreviation: cerf_state_id}

  • expansion_plan_xml_dir – Full path to the directory to save the expansionplan.xml file to.

  • output_csv – Boolean. Choose to save a copy to of the expansion plan as a CSV file. Default is False.
